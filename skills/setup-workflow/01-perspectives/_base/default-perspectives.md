# Setup Workflow 視角定義

本 Skill 為配置型 Skill，不使用多視角分析模式。

## 說明

`setup-workflow` 是一個一鍵配置工具，用於設置 multi-agent-workflow 的規範執行機制。

由於這是一個配置操作而非研究分析任務，因此不需要多視角協作。

## 配置項目

| 配置 | 說明 |
|------|------|
| Hook 腳本 | 自動 commit、驗證檢查 |
| 權限預設 | 常用命令預先允許 |
| Memory 結構 | 研究/計劃/實作記錄目錄 |
