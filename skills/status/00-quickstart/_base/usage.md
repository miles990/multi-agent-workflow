# Quick Start Guide

> 1 分鐘快速上手 Status

## 最簡用法

```bash
/status
```

這會：
1. 讀取當前工作流狀態
2. 顯示階段進度和任務列表
3. 呈現視覺化的進度條和狀態圖標

## 常用模式

### 查看當前工作流

```bash
/status
```

顯示當前執行中的工作流狀態。

### 查看特定工作流

```bash
/status user-auth
```

顯示指定 workflow-id 的狀態。

### 列出歷史記錄

```bash
/status --list
```

列出所有工作流歷史。

### 顯示任務 DAG

```bash
/status --dag
```

生成 Mermaid 任務依賴圖。
